<h3>General Info</h3>
<p>Flash-based powerful chat solution for social sites with many features, such as: audio/video broadcastings, private conversations, chat history and others.</p>
<h3>No Media Server</h3>
<p>Chat uses free Adobe Cirrus service for media connections that go directly from one user's application to the other one's without any media server between them. That makes media connection very fast and really real-time.</p>
<h3>Audio/Video Chat</h3>
<p>Users have an ability to share their video and audio with any other users in the chat. It is possible to deny video playing from undesired watchers. User can watch several videos in the moment with one video enlarged into popup window.</p>
<h3>Easy administration</h3>
<p>Login Chat application under admin account to gain admin priviligies. Admin can ban/unban, kick users, set chat moderators, manage admin settings and provide different users groups with certain rights and permissions.</p>
<h3>General Features</h3>
<p>All general chat features are available: text-chatting, smileys support, quick messages, online status, font customizing, multi-rooms support, chat history, passowrd-protected rooms, private conversations, files transferring, audio/video chat and others.</p>
<h3>Chat History</h3>
<p>All chat conversations are stored in database and can be easily found by admin at any time by chat date.</p>
<h3>Private Conversations</h3>
<p>Users can talk between each other privately with audio/video and text chat like they talk via messenger.</p>



<h3>Installation and configuration:</h3>

1. Copy the rzchat plugin folder into the mod folder on your server,
2. Activate the plugin in the admin section of your site,

Login front-end of your site with admin account, open <a href="http://www.rayzzz.com/docs/chat/settings" target="_blank">settings</a> and enter <a href="http://www.rayzzz.com/docs/adobe-key" target="_blank">Adobe Labs Developer Key</a>.
Additional configuration: for the chat session cleanup to work Elgg's hourly cronjob must be set up on the server.
