/**
 * Description
 */
Enables a custom topbar logo for your Elgg site.

Administrators are able to upload/remove a custom topbar logo.

Will display the exact file without resizing the image - so make sure 
you got the appropriate resolution. Allowed mimetypes are 'image/gif', 
'image/jpg', 'image/jpeg', 'image/pjpeg' and 'image/png'.

No feature of setting height and width.. will just be implemented on request.
